FILGRIMAGE

Filgrimage는 성지순례라는 뜻으로
인터넷상에서는 자기 커리어에서 성공한 사람의
과거 커리어를 찾아 들어오거나 앞으로 잘 될것 같은
사람들을 예측하는 것을 말합니다.
Filgrimage는 국내외 영화제에서 주목받고 있지만
대중들에게 잘 알려지지 않은 독립영화들을
소개합니다. 독립영화가 낯선 사용자들의 진입장벽을
낮추기 위해 독립영화에서부터 차근차근 커리어를
쌓으며 주목을 받아 현재의 네임벨류를 얻게된
배우들의 필모그래피를 중심으로 소개해서 스타들의
과거 색다른 모습을 소개하며 사용자의 흥미를
이끌어 냅니다.
